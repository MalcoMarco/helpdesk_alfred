FAQ

FAQ is a feature where define lots of questions and answers that customer might have. On the Dashboard you would be able to set those question and answers easily. You will be able find the FAQ manage page on the following admin link. http://website.com/dashboard/faqs

List

FAQ List
You will also find the faq list page if you click in the FAQs menu from the left sidebar section.

Create

Create a new FAQ
You would be able create a new FAQ item if you click on the Create New button on the FAQ list page.
Fill out the required fields and click on the Create FAQ button to create a new FAQ item.

Update

Update FAQ
You can update an existing FAQ item easily with select/click on any FAQ item from the FAQ list page.

Delete

Delete a FAQ Item
Go to the FAQ details page with clicking on a particular FAQ item. Then click on the "Delete" button to delete that particular FAQ.
