Blog

You would be able to manage your blog posts here which will be appear on your front/public site. You will be able find the blog posts on the following admin link. http://website.com/dashboard/posts

Post List

Blog Post List
You will also find the post list page if you click in the Blog menu from the left sidebar section.

Create

Create a new post
You would be able create a new post item if you click on the Create a new post button on the post list page.

Update

Update a post item
You can update an existing post item easily with select/click on any post item from the post list page.

Delete

Delete a blog post
Go to the Post details page with clicking on a particular post item. Then click on the "Delete Post" button to delete that particular Post.
