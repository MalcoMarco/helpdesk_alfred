Contacts

Contacts is an additional module where you can add your essential contacts that you need. You would be able to manage your contacts here. You will find that contacts page on the following URL. http://website.com/dashboard/more/contacts

List

Contact List
You will find the contact list page if you click in the Contacts menu from the left sidebar section.

Create

Create a new contact
You would be able create a new Contact item if you click on the Create New button on the Contact list page.

Update

Update contact
You can update an existing contact item easily with select/click on any contact item from the contact list page.

Delete

Delete a contact
Go to the contact details page with clicking on a particular Contact item. Then click on the Delete Contact button to delete that particular contact.
