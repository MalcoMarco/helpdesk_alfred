Customers

You would be able to manage your customers on this page. You will find that customers page on the following URL. http://website.com/dashboard/customers

List

Customer List
You will find the customer list page if you click in the Customers menu from the left sidebar section.

Create

Create a new customer
You would be able create a new customer item if you click on the Create New button on the customer list page.

Update

Update a customer item
You can update an existing customer item easily with select/click on any customer item from the customer list page.

Delete

Delete a customer
Go to the customer's details page with clicking on a particular customer item. Then click on the Delete button to delete that particular customer.
